# 🌐 Google Homepage Clone (HTML & CSS)

Este proyecto consiste en la creación de un clon visual de la página principal de Google utilizando únicamente **HTML y CSS**, con el objetivo de practicar maquetación, estructura semántica y diseño de interfaces modernas.

Actividad desarrollada para el curso de "Programación orientada a objetos y base de datos relacionales" del SOC-CIEF. Impartido por el formador Óscar Burgos. 

---

## 🎯 Objetivo

Reproducir la interfaz de Google lo más fiel posible, prestando atención a:

* Distribución de elementos (layout)
* Uso de Flexbox
* Detalles visuales (espaciado, tamaños, colores)
* Interacciones básicas (hover, focus)
* Buenas prácticas en HTML y CSS

## 🧠 Conceptos aplicados

### 📌 HTML

* Uso de etiquetas semánticas: `header`, `nav`, `main`, `section`, `footer`
* Formularios (`form`, `input`, `button`)
* Organización en bloques lógicos

### 🎨 CSS

* Flexbox para layout
* Reset básico (`box-sizing`, `margin`, `padding`)
* Estados interactivos (`:hover`, `:focus-within`)
* Tooltips personalizados
* Estilizado de botones y enlaces
* Control del comportamiento del input

---

## 🔍 Funcionalidades implementadas

* Barra de navegación (header)
* Buscador central con iconos
* Botones “Buscar con Google” y “Voy a tener suerte”
* Tooltips al pasar sobre iconos
* Estilizado tipo Google del input (sin borde azul por defecto)
* Footer con enlaces organizados
* Botón “Modo IA” con estilo moderno

---

## 💡 Aprendizajes clave

* La estructura es más importante que el diseño
* Flexbox controla el comportamiento de los hijos, no de los nietos
* No todo lo que aparece en pantalla lo controla tu código (ej: autocompletado del navegador)
* Un pequeño error de sintaxis puede romper todo el CSS
* Debuggear es parte esencial del desarrollo
